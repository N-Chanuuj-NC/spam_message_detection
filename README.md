# Spam Detection System

## 📌 Description
This project classifies messages as spam or not spam using Natural Language Processing techniques.

## 🚀 Features
- Text vectorization using TF-IDF
- Naive Bayes classification
- Real-time message prediction

## 🛠 Technologies
- Python
- Scikit-learn

## ▶️ How to Run
python src/main.py
